# REFLA — зеркала на входные двери

## Запуск

1. Установите зависимости: npm i
2. Режим разработки: npm run dev
3. Сборка и экспорт: npm run build && npm run export (статический сайт в out/)

Хостинг без Node.js: загружайте содержимое папки out/ и файл lead.php уже находится на сервере. Форма отправляет JSON на /lead.php.

## Страницы

О нас / Примеры / Прайс-лист / Контакты / Оставить заявку

## Форма

POST /lead.php с JSON: { name, contact, address, comment }. Ответ ожидается JSON: { ok: true }.
